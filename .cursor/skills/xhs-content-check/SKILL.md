# 小红书发帖前内容检测

## 任务

收到帖子内容（标题 + 正文 + hashtag）后，按以下三层逐一扫描，输出结构化检测报告。

---

## 检测层级

### 🔴 Layer 1：违禁词（发帖即删/封号）

以下任一词出现，直接标记为"高危，必须删除"。

**平台引流类**
- 微信号、QQ号、手机号（直接出现原始号码）
- 加我、V我、私信我（引导站外联系）
- 二维码（配合引导扫码的语境下高危）
- 链接、网址、http、www
- 淘宝、京东、拼多多、闲鱼、得物（直接引导去他平台购买）
- 抖音、微博、B站（引导去竞品平台）

**绝对化/极限词（广告法）**
- 最好、最强、最优、最便宜、最低价、最高、最新
- 第一、No.1、排名第一、销量第一、全网第一
- 唯一、独家、独创
- 顶级、顶尖、极品、终极

**虚假承诺**
- 100%、保证、必然、一定（用于效果承诺时）
- 包退、包过、稳赚不赔、零风险
- 立竿见影、马上见效、一天见效

**医疗健康类**（非医疗产品禁用）
- 治疗、治愈、根治、消炎、杀菌
- 祛痘、淡斑、美白、抗衰（功效宣称时违规）
- 医用、药用、医学级、皮肤科医生推荐

**诱导互动（违规）**
- 一键三连、点击领取、点击有惊喜
- 评论抽奖（"评论XX抽XX"格式需谨慎）

---

### 🟡 Layer 2：限流词（降权/减少曝光，不删帖）

出现时标记"中风险，建议替换"，并给出替换词。

| 限流词 | 建议替换 |
|--------|---------|
| 购买 | 入手、带走 |
| 下单 | 种草了、决定试试 |
| 价格 / 多少钱 | 多少米、性价比 |
| 收费 | 收了一点点 |
| 广告 / 推广 | （直接删除） |
| 接单 | 在接了、约上了 |
| 约拍 | 找了个拍照的（伪素人帖中） |
| 私信 | 评论区告诉我 |
| 加群 | （避免出现） |
| 免费 | 白嫖、不要钱（非正式语境） |
| 史上最低价 / 全网最低 | 超划算、很便宜 |
| 限时抢购 / 秒杀 | 今天试了 |
| 正品 | 真的好用 |
| 返利 / 佣金 | （删除） |
| 回购无数次 | 用了很多次 |
| 效果显著 | 感觉还不错 |

---

### 🔵 Layer 3：像素拍档专项检查（账号风险）

针对旅拍伪素人帖的特殊场景检查：

**1. 伪素人暴露风险**
- 直接说"我们平台" / "欢迎入驻" / "官方账号" → 素人身份暴露，高危
- 语气过于商业化："完美体验""专业服务""一站式解决" → 降权

**2. 引流方式检查**
- "小程序" 出现 → 安全（小红书允许提到微信小程序）
- "像素拍档" 出现 → 安全（平台名不违规）
- "二维码" 在引导扫码语境中 → 建议改为"那个码"
- "扫码" 单独出现 → 安全；"扫码+加我" 组合 → 高危

**3. 价格表述检查**
- 出现具体金额（如"20元/张"）→ 中风险，改为"几十块""很划算"
- 出现比价描述（"比XXX便宜"）→ 避免拉踩竞品名称

---

## 输出格式

```
## 🔴 高危（必须修改）
[每个高危词 → 所在位置 → 具体替换方案]

## 🟡 中风险（建议修改）
[每个限流词 → 替换词]

## 🔵 素人账号专项
[伪素人暴露风险 / 引流方式问题]

## ✅ 总评
- 风险等级：高危 / 需调整 / 可发布
- 需修改处：X 处

## 📝 修改后版本（只改问题词，不重写全文）
[替换后的完整帖子文本]
```

---

## 注意事项

1. **谐音梗不安全**："薇信""V❤"等变体平台已能识别，处罚更重
2. **hashtag 同样检测**：话题标签中的限流词也影响曝光
3. **图片内文字**：平台 OCR 会扫图片内容，封面文字也要检查
4. **矩阵号节奏**：同批账号短时间发类似内容会触发检测，建议间隔 ≥1天
